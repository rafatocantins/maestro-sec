# MaestroSec (Secure Multi-Agent Orchestration Platform)

## Overview
MaestroSec is a "master agent" platform that plans and governs a fleet of agents executing tasks with Zero Trust, auditability, and clear boundaries.

## Architecture
- **Master Orchestrator**: Plans and delegates tasks.
- **Fleet Agents**: Execute tasks on endpoints (Linux/Windows/macOS).
- **Policy Engine**: Enforces RBAC/ABAC and extensive rules.
- **WhatsApp Gateway**: Interface for command and control.

## Project Structure
- `apps/ui`: Next.js Dashboard
- `services/`: Backend microservices
- `agents/`: Fleet agent implementation
- `infra/`: Infrastructure as Code
- `docs/`: Documentation and ADRs
